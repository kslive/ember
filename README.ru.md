# Ember

**Локальная запись, расшифровка и саммари встреч для macOS.**

Ember записывает микрофон и собеседника, ведёт живой транскрипт и превращает его в
подробное, читабельное саммари — целиком на вашем Mac. Стартует и останавливается сам, и
по умолчанию ничего не покидает устройство.

## Что это

Ember записывает встречу (микрофон + системный звук), расшифровывает её на устройстве и
пишет повествовательное саммари, которое можно прочитать вместо присутствия: разделы по
темам с деталями, именами и решениями прямо в тексте, плюс дальнейшие шаги. Работает офлайн
и без аккаунта; единственная облачная опция — саммари через DeepSeek, выключена, пока вы не
добавите ключ.

## ✨ Возможности

### 🎙 Запись и захват
- **Обе стороны звонка** — ваш микрофон и системный звук (те, с кем вы говорите) пишутся
  вместе, в том числе через **AirPods / наушники**.
- **Автодетект звонка** — запись стартует через несколько секунд после начала звонка (Zoom,
  Google Meet, встречи в браузере, …) и останавливается по его окончании, даже когда Ember
  свёрнут.
- **Управление из меню-бара**, **⌘R** для старта/стопа и понятные этапы обработки после звонка.

### ⌨️ Живой транскрипт
- **Транскрипт в реальном времени**, печатается по мере речи.
- **Метки спикеров** — `[Я]` для вас и `[С]` для собеседника; кто именно из участников
  говорит, ИИ разбирает из содержания разговора.
- **Два движка распознавания** — Whisper (Small / Medium / **Large V3 Turbo**, мультиязычный)
  и **GigaAM v3**, в 2–3 раза точнее для русского.

### 🧠 Саммари
- **Повествовательное саммари на языке транскрипта** — читаете вместо присутствия на встрече.
- **По умолчанию локально** через Apple **MLX (Qwen3 1.7B / 4B / 8B)**, либо добавьте
  опциональный **ключ DeepSeek API** — почти мгновенные облачные саммари с **автоматическим
  фолбеком** на локальную модель.
- **Опора на факты** — имена, числа и решения берутся из реально сказанного, меньше
  выдумок.
- **Правка на лету** — исправляйте саммари прямо при чтении; изменения синхронизируются с
  экспортированным файлом.

### 🗂 Шаблоны саммари
- **Выбирайте, как пишется каждое саммари** — задайте дефолт или свой шаблон для конкретной
  встречи.
- **Встроенная библиотека**: Standard, **Daily, Interview, 1×1, Final Interview, TODO, Demo,
  Grooming**.
- Шаблоны — обычные **Markdown-файлы** в папке: правьте руками или кидайте свои, изменения
  подхватываются автоматически. Каждая встреча помнит шаблон, которым сделана.

### 📅 Порядок и экспорт
- **Заголовки из календаря** — встреча называется по событию Apple Calendar, которое идёт в
  момент начала записи.
- **Поиск** по всем встречам и свободное переименование.
- **Экспорт в Markdown** по папкам дат (YAML-фронтматтер, задачи, тайм-коды).
- **Открытие в Obsidian одной кнопкой** — или в **Sage**, который забирает кнопку себе, если
  установлен.

### ⚙️ Управление и полировка
- **Отложенная обработка** (опционально) — если сразу идёт следующий звонок, расшифровка и
  саммари ждут в очереди и запускаются, когда все звонки закончились: Mac не шумит во время
  разговора.
- **Управление моделями** — скачать / выбрать / удалить, живой прогресс, подсказки по
  памяти, сгруппированы и отсортированы; модели выгружаются сразу после использования,
  освобождая память.
- **Запуск при входе в систему**, светлая / тёмная / авто тема, **шесть акцентных цветов** и
  **русский / английский / китайский** с мгновенным переключением.
- **Встроенные обновления** из GitHub Releases: плашка обновления и окно «Что нового».

## Экспорт в Obsidian

Каждое саммари можно сохранить в Markdown-файл (YAML-фронтматтер, задачи, тайм-коды) в
выбранную папку и открыть в **Obsidian** одной кнопкой. Если установлен
**[Sage](https://github.com/kslive/sage)** — приоритет у него: кнопка превращается в светящуюся
**«Открыть в Sage»**, и заметка открывается прямо в Sage (при необходимости он сам переключит
пространство на папку заметки). Удалите Sage — вернётся кнопка Obsidian.

## Установка

Нужна **macOS 14.4+** на **Apple Silicon**.

1. Скачайте `Ember_1.6.0_aarch64.dmg` со страницы [Releases](../../releases/latest).
2. Перетащите **Ember.app** в **Программы**.
3. Приложение подписано ad-hoc (без нотаризации), поэтому первый запуск блокируется.
   Либо правый клик по **Ember.app → Открыть → Открыть**, либо в Терминале:
   ```bash
   xattr -dr com.apple.quarantine /Applications/Ember.app
   ```
4. При первом запуске выберите языки, затем скачайте модель Whisper и модель для саммари.

## Сборка из исходников

Нужны свежий Xcode и [Tuist](https://tuist.dev).

```bash
cd native
tuist install
tuist generate
open Ember.xcworkspace          # или: xcodebuild -scheme Ember -configuration Release build
```

Модели подтягиваются с Hugging Face при первом запуске. Подпись только ad-hoc — аккаунт
Apple Developer не нужен.

## Приватность

- Запись, расшифровка и саммари выполняются локально.
- Аудио пишется во временную папку и удаляется сразу после расшифровки; сохраняется только
  текст (транскрипт + саммари) в локальной базе SQLite на вашем Mac.
- Саммари по умолчанию локальные. Если добавить ключ DeepSeek API (опционально), транскрипт
  отправляется в DeepSeek для суммаризации; удалите ключ — и всё снова полностью локально.
  Ключ хранится на вашем Mac в зашифрованном виде.
- Кроме этого, единственный выход в сеть — загрузка моделей (Hugging Face) и проверка
  обновлений (GitHub). Без телеметрии и аккаунтов.
