# Ember

**Локальная запись, расшифровка и саммари встреч для macOS.**

Записывает микрофон и собеседника, ведёт живой транскрипт с метками спикеров и
делает подробное саммари — на вашем Mac. По умолчанию ничего не выгружается в сеть.

## Что это

Ember записывает встречу (микрофон + системный звук), расшифровывает её на устройстве
и пишет повествовательное саммари, которое можно прочитать вместо присутствия: разделы
по темам с деталями, именами и решениями прямо в тексте, плюс дальнейшие шаги. По
умолчанию работает офлайн и без аккаунта; единственная облачная опция — саммари через
DeepSeek, выключена, пока вы не добавите ключ.

## Возможности

- **Живой транскрипт** во время записи с метками спикеров: `[Я]` и собеседники — разные
  голоса другой стороны различаются и нумеруются (`[С1]`, `[С2]`).
- **Автодетект звонка** — запись стартует через несколько секунд после начала звонка
  (Zoom, встречи в браузере, …) и останавливается по его окончании, в том числе когда
  окно свёрнуто.
- **Саммари на языке транскрипта**: локально через Apple MLX (Qwen3), либо добавьте
  опциональный **ключ DeepSeek API** — почти мгновенные облачные саммари с автоматическим
  фолбеком на локальную модель.
- **Два движка распознавания**: Whisper (мультиязычный) и **GigaAM v3** — в 2–3 раза
  точнее для русского.
- **Поиск** по всем встречам, переименование и экспорт в Markdown / Obsidian.
- **Меню-бар**, **⌘R** для старта/стопа и видимые этапы обработки после звонка.
- Светлая / тёмная / авто тема, шесть акцентных цветов; русский, английский и китайский.
- Встроенные обновления из GitHub Releases: плашка на главном экране и окно «Что нового».

## Экспорт в Obsidian

Каждое саммари можно сохранить в Markdown-файл (YAML-фронтматтер, задачи, тайм-коды) в
выбранную папку — например, в волт Obsidian.

## Установка

Нужна **macOS 14.4+** на **Apple Silicon**.

1. Скачайте `Ember_1.5.1_aarch64.dmg` со страницы [Releases](../../releases/latest).
2. Перетащите **Ember.app** в **Программы**.
3. Приложение подписано ad-hoc (без нотаризации), поэтому первый запуск блокируется.
   Либо правый клик по **Ember.app → Открыть → Открыть**, либо в Терминале:
   ```bash
   xattr -dr com.apple.quarantine /Applications/Ember.app
   ```
4. При первом запуске выберите языки, затем скачайте модель Whisper и модель для саммари.

## Сборка из исходников

Нужны свежий Xcode и [Tuist](https://tuist.dev).

```bash
cd native
tuist install
tuist generate
open Ember.xcworkspace          # или: xcodebuild -scheme Ember -configuration Release build
```

Модели подтягиваются с Hugging Face при первом запуске. Подпись только ad-hoc — аккаунт
Apple Developer не нужен.

## Приватность

- Запись, расшифровка и саммари выполняются локально.
- Аудио пишется во временную папку и удаляется сразу после расшифровки; сохраняется только
  текст (транскрипт + саммари) в локальной базе SQLite на вашем Mac.
- Саммари по умолчанию локальные. Если добавить ключ DeepSeek API (опционально),
  транскрипт отправляется в DeepSeek для суммаризации; удалите ключ — и всё снова
  полностью локально. Ключ хранится на вашем Mac в зашифрованном виде.
- Кроме этого, единственный выход в сеть — загрузка моделей (Hugging Face) и проверка
  обновлений (GitHub). Без телеметрии и аккаунтов.
